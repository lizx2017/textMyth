Summary:
If you make custom item renderer for each column in an ADG then dragProxy on dragging a row does not show the labels present in different columns.
Description:
Steps to reproduce:
1. Create an ADG with custom item renderer for each column.
2. Now drag a row.
3. The drag proxy that is made by DragManager does not show the labels present in different columns.
Actual Results:
DragProxy does not show the labels present in different columns.
Expected Results:
It should show the all the labels of that row like it does if you do not make custom item renderer for columns.
