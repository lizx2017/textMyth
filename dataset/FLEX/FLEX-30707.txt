Summary:
Selecting item using space should select only that item
Description:
Reproduction:
1. Have a chart with multiple selection set to true (or run bug file and use any chart)
2. select one item
3. pressing CTRL key move to some other item using arrow keys
4. release the CTRL key and press SPACE
Actual Results:
The earlier selected item remains selected
Expected Results:
The earlier selection should clear and only item selected using space should remain selected
