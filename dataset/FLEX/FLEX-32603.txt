Summary:
In AdvancedDataGrid with hierarchical data, inserted rows causes rows inaccessable through keyboard
Description:
Steps to reproduce:
1. add a child node to a hierarchical data structure in and ADG.
2. use keyboard 'down arrow' to move the selected row.
3. the last row is not reachable using the keyboard.
4. if 'n' rows were inserted, then the last 'n' rows become unreachable.
5. all rows are selectable through mouse gestures.
Actual Results:
Expected Results:
All rows are selectable through the keyboard
Workaround (if any):
After a folder is collapsed and re-expanded, all rows become selectable through the keyboard.
