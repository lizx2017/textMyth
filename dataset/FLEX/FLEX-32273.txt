Summary:
Rollover state not cleared in Advanced Datagrid when some other item is selected
Description:
Steps to reproduce:
1. Create an advanced datagrid with multiple rows
2. Select a row. It gets highlighted.
3. Now select a different row.
Actual Results:
1. The previous row that was selected is still highlighted.
2. Also the rollover doesn't highlight the the rows on mouse over when mouse is rolled randomly over different rows
Expected Results:
1. Rollover should be cleared when some other item is selected.
2. The row on which mouse is rolled should get highlighted and it should change accordingly.
