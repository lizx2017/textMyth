Summary:

Description:
this function doesn't work in 0.8 dev.
<fx:Script>
  <![CDATA[
  function foo():void
  {
   var a: Array = new Array("a", "b", "c");
   a.removeAt(2);
  }
  
  ]]>
 </fx:Script>

testlang07dec06.prototype.foo = function() {
  var /** @type {Array} */ a = new Array("a", "b", "c");
  a.removeAt(2);
};
but in earlier FlexJS sdk, it was built to
a.splice(2,1);
