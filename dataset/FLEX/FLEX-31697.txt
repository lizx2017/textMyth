Summary:
Resizing bubble charts corrupts the display
Description:
Steps to reproduce:
1. Create a normal bubble chart and dont set maxRadius property
2. or alternatively run bug file
3. resize the application window to smaller size and resize it back to normal size
Actual Results:
Bubble chart display is corrupted, numeric axis is lost
Expected Results:
Bubble chart should be shown properly
Workaround (if any):
set maxRadius property to 20 on BubbleChart
