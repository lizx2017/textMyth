Summary:
Callout allows point upside down on three clicks
Description:
Found in 22957 and air 468
Steps to reproduce on android
1. Run the attached project
2. Select "Callout Button" view
3. Turn to landscape (also happens on portrait)
4. Click on any of Callout buttons "thrice" quickly
Actual Results:
The callout arrow points downwards
Expected Results:
Callout arrow should point to its owner component
Steps to reproduce on iOS: (IpodTouch)
1. Run the attached project
2. Select "Callout Button" view
3. Turn to landscape (also on portrait)
4. Click on any of Callout button once to open callout
5. Click twice quickly on the same button that has a open callout
Actual Results:
The callout arrow points downwards
Expected Results:
Callout arrow should point to its owner component
