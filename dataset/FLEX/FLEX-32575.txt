Summary:
Advanced Data Grid Open/Close Default Animation Not Functioning
Description:
Steps to reproduce:
1. Create an advanced data grid with children nodes
2. Run application
3. Open one of the nodes on the advanced data grid
Actual Results:
Default open animation is not playing as with Flex 3.
Expected Results:
Default open animation should play like default settings in Flex 3.
Workaround (if any):
Explicitly set OpenDuration property to 250 (this should be the default according to the documentation for Flex 4).
