Summary:
If arrow keys, PgUp / PgDn, Home / End or SPACE are used with the SHIFT key unknown behavior happens
Description:
Reproduction:
1. Open bug file
2. In area chart sample select second item in amount series(blue color)
3. Holding SHIFT pressed use PgUP/PgDn. HOME/END or arrow keys

Unknown behavior happens where items from different series get selected. Sometimes multiple items from multiple series get selected with just one arrow button hit.
Expected Results:
If the series has a anchor item, all the items between the anchor item and the item corresponding to PgUp / PgDn, Home / End are selected
