Summary:
Printing support for multipage RicheditableText HTML text
Description:
Steps to reproduce:
1. Create a print view that have Richeditabletext control that will take in multipage HTML text
2. Print the printview
Actual Results:
Content in Richeditabletext Control got cut at page end. No Header/Footer or page number can be added anywhere.
(Very obvious if there is a picture at end of page. The picture could get cut in half at times.)
Expected Results:
Should be able to print the multipage Richeditabletext content with header/footer page number and content not cut off at end of page.
