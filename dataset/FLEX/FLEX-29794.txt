Summary:
AIR PLUGIN Issue -Very low level of control on the pointing hand movements if u use object spy and move over flex components in the Air application
Description:
Steps to reproduce:
1.Open a test AIR Application.
2.Click on Tools->Object Spy within QTP.
3.Using the pointing hand click on a flex component within the application.
Actual Results:Very low control on the pointing hand and it is very difficult to select a desired flex object.
Expected Results:You should be able to move the pointing hand over the flex objects with ease.
Workaround (if any):Click on the control button and move the pointing hand to the desired flex component ,release the control button and then select the object .
Please note ;This issue is with respect to the new AIR Plugin.
