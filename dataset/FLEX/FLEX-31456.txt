Summary:
Refresh issue while resizing the "tree" type ADG in design view
Description:
Steps to reproduce:
1. Drag & Drop ADG from Components panel to Design view.
2. Select type of data as "tree" and click on "OK" button.
3. Click and Drag the corners of ADG to resize.
Actual Results:
ADG Headers and Borders are not displayed properly. ( Please see the attached screen shots for better understanding of the issue )
Expected Results:
ADG should be displayed properly even after resizing.
Workaround (if any):
Click on "Refresh" icon to display properly.
Note: This resizing issue DOES NOT occur when type of data is "flat"
