Summary:
CSS Design View is not refresh when 'Horizontal Grid Lines' property is unchecked
Description:
Steps to reproduce:
1. Create a CSS file in your Flex project and go to CSS Design View
2. Switch to Design tab in CSS Design view and Add a style for AdvancedDataGrid { }
3. Check the 'Horizontal Grid Lines' checkbox in PI and observe the change in preview.
4. Now, uncheck 'Horizontal Grid Lines'.
Actual Results:
CSS Design preview does not refresh
Expected Results:
CSS Design preview should get refreshed automatically.
Workaround (if any):
Click on "Refresh" button.
Note: 'Vertical Grid Lines' checkbox refreshes the preview fine.
