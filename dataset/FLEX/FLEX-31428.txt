Summary:
when setting openItems for a Tree Data Grid, the nodes get opened, but it doesn't close already open nodes.
Description:
Steps to reproduce:
1. run the bugfile.

3. close one node on the first ADG and again click the button.
Actual Results: The closed node doesn't get reflected on the second ADG
Expected Results: It should get reflected on the second ADG.
Workaround (if any): Reset the dataProvider and then set open nodes.
