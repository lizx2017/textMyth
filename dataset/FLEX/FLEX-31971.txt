Summary:
AdvancedDataGrid sort by clicking on the header does not work correctly
Description:
Steps to reproduce:
1. Click on the header for any column. Notice that the sort works
2.Click again on the same column header. The sort order does not change. Expected behaviour is for it to change.
Actual Results:
Clicking on the column header second time around does not reverse the sort order
Expected Results:
Clicking on the column header second time around should reverse the sort order
Workaround (if any):
Ctrl+click on the column header second time around does reverse the sort order
