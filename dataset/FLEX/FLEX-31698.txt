Summary:
Clicking on the second sort field to make it as the first sort field does not work
Description:
Steps to reproduce:
1. Compile & Launch the attached MXML file
2. Click on the first column to make it as first sort field.
3. Click on the second column to make it as the second sort field.
4. Click on the center of the second column.
Actual Results:
Clicking on the center area of the second column's header does NOT make it as the first sort field.
Expected Results:
Clicking on the center area of the second column's header SHOULD make it as the first sort field.
Workaround (if any):
Click on third column to make it as a the first sort field and then click on the second column to make it as first sort field.
Note : This is a must fix for this release.
