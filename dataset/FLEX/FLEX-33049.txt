Summary:
Legend Marker disappears when the data provider is changed on the chart
Description:
Steps to reproduce:
1. Create a pie chart with a legend.
2. Switch data provider.
Actual Results:
1. Chart updates.
2. Legend Labels update.
3. Legend Markers disappear.
Expected Results:
1. Chart updates.
2. Legend Labels update.
3. Legend Markers update.
Workaround (if any):
Reset the data provider on the Legend once the chart has finished updating (updateComplete event). Please see attachment.
