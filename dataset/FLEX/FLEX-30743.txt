Summary:
CTRL and range selection doesnt preserve previous selection
Description:
Reproduction:
1. Have a chart with multiple selection mode
2. Click on any chart item and select it
3. Pressing CTRL select some other chart items using range selection
Actual Results:
Only items under the range selection are selected
Expected Results:
Previous selection should be added to items under range.
WorkAround:
Dont use range selection insteat click on item when CTRL is pressed
