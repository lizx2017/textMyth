Summary:
unable to paste text if Malayalam is the system's input source
Description:
User is unable to paste text if Malayalam is set as the system's input source.
Steps to reproduce:
Open an AIR application with a text input or text area.
Select "Malayalam" as input source (in OS)
Type some text in the text input, select all the text and cut it (CMD-X)
Paste the text (CMD-V)
-> Nothing happens
Expected behaviour:
User should be able to paste Malayalam text.
Tested on different machines. Issue is 100% reproducible.
Note: Happens in MAC OSX and AIR only. In Windows and Web issue is not reproducible.
