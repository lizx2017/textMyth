Remove bundling of Apache 1.0 licensed software
Description:

The Flex SDK use code from the Apache JServ project which is Apache 1.0 licensed. [2]
1. https://www.apache.org/licenses/LICENSE-1.0
2. https://github.com/apache/flex-sdk/blob/develop/modules/thirdparty/velocity/test/texen-classpath/jar-contents/license.txt
