Summary:
Blank line assumed to be bullet point if it precedes a bullet point list
Description:
Steps to Reproduce
1. Open a Rich Text Editor; in the text area hit enter to go to a new line
2. Create a bullet-point list
3. Save the contents as html
Expected Behavior
In the html produced, there should be a blank paragraph followed by an html list
Actual Behavior
There is no blank paragraph; there is instead just a bullet point list with the first list item being blank.
