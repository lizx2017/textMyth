Summary:
Column Sort Arrow on AdvancedDataGrid switches direction when mouse hovers over column heading
Description:
Steps to reproduce:
1. Create a Flex app with AdvancedDataGrid.
2. Sort on a column. Should be in ascending order with arrow pointing up.

4. Move mouse back to column heading as if to sort again
Actual Results:
Sort arrow switches from down-arrow to up-arrow
Expected Results:
Sort arrow should remain as down-arrow.
