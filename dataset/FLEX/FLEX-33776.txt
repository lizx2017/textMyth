Summary:
Software keyboard prefill TextInput when displayAsPassword is set true and spark.skins.mobile.TextInputSkin is used!
Description:
When software keyboard is opened in TextInput and "Predictive text" feature on device is switched on, after entering some symbols in first field and focus another TextInput (displayAsPassword=true) without hiding keyboard and start writing - last entered symbols are automatically filled in this field!
On both fields is applied spark.skins.mobile.TextInputSkin skin.
