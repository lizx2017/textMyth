Summary:
ADGGroupItemRenderer doesn't word wrap when wordWrap is set to true on a column
Description:
Steps to reproduce:
1. Create a ADG with hierarchicaldata with long text labels and variableRowHeight set to true.
2. Set wordWrap to true on the first column
3. Resize the column
Actual Results:
ADGGroupItemRenderer doesn't wordWrap.
Expected Results:
ADGGroupItemRenderer should wordWrap.
Workaround (if any):
Set wordWrap on the ADG itself.
