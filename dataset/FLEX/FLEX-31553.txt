Summary:
Cell navigation into summaryRow throught keyboard highlights just the cell rather than the entire summaryRow
Description:
Steps to reproduce:
1. Compile and run the bugfile.
2. Click the Grouping button to apply grouping and the summary.
3. Click on any cell on say the second column and using arrow keys navigate up to the summaryRow
Actual Results: Only the cell is highlighted instead of the entire row
Expected Results: The entire row should be highlighted
Workaround (if any):
1. Mouse navigation works fine.
2. Keyboard navigation works fine on clicking on any cell in first column and then moving up. It is erroneous in all other columns
