Summary:
Incorrect color space processing for inline images
Description:
Getting wrong color space for inline images because of abbreviations. (PDF Reference 1.4, chapter 4.8.6).
According to specification inline images can use directly (not by names) only 4 color spaces: DeviceGray (Gray), DeviceCMYK (CMYK), DeviceRGB (RGB), Indexed (I).
